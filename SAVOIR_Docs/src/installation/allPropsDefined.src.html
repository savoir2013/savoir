Introduction

This document details all the build properties found in the file /pathToSavoir/savoir-${savoir.release.ver}/src/SAVOIR_MasterBuild/localDefault.properties. It mostly consists of a table listing each property along with its description. The properties are presented in the same order that they appear in the file.

SAVOIR Build Properties

The SAVOIR build infrastructure supports deployment to both a production server and a development server. The mechanism to implement this dual deployment system is through the use of properties. The properties file localDefault.properties contains all the properties that need to be defined to build SAVOIR.

The following table lists all the build properties along with a brief description for each property. Given that the file provides for both a production and development environment, many of the properties seem to be duplicated. However, they are listed in the table so that they can easily be matched to the properties within the localDefault.properties file.
